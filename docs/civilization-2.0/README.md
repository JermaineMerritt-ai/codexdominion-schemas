- **Youth** - Inheritors in training (becoming)
- **Creator** - Builders and innovators (building)
- **Legacy-Builder** - Stewards and mentors (stewarding)

**Seasonal Rhythm:**
- **Dawn** (Vision) - Set goals, define missions
- **Day** (Action) - Execute, build, contribute
- **Dusk** (Reflection) - Review, refine, learn
- **Night** (Renewal) - Rest, reset, prepare

**Core Symbols:**
- **The Flame** - Continuity, inheritance, sovereignty
- **The Circle** - Unity, community, identity cycles
- **The Crown** - Stewardship, not domination
- **The Seasons** - Operational rhythm
- **The Bridge** - Diaspora connection
