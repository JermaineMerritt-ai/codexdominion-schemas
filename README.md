# CodexDominion — Minimal Constellation Setup

CodexDominion is a civilization‑grade monorepo uniting apps, packages, agents, and core governance.

## Structure
- `core/` → Council Seal, Sovereigns, Custodians, Flows
- `apps/` → Chatbot, Commerce, Observatory, Compliance
- `agents/` → Avatars for healthcare, legal, commerce, cybersecurity
- `packages/` → Shared libraries (UI, utils, schemas, healing)
- `customers/` → Customer portal

## Setup
```bash
npm install
npm run build
npm test
```

## Standards
- Variables → camelCase
- Classes → PascalCase
- Directories → kebab-case
- Commits → Conventional Commits
