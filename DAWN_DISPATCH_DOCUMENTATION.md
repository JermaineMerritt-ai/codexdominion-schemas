#### Common Issues

1. **"No authentication credentials provided"**
   - Normal for unconfigured services
   - Reports continue with available data
   - Configure service APIs to eliminate warnings

1. **"Systems CRITICAL" status**
   - Indicates low health score (< 50%)
   - Check individual service connections
   - Verify API credentials and network connectivity

1. **Archive file errors**
   - Check file permissions in working directory
   - Verify disk space availability
   - Ensure JSON syntax in existing archives

1. **Scheduler not running**
   - Verify schedule configuration syntax
   - Check system clock and timezone settings
   - Monitor scheduler log file for errors
