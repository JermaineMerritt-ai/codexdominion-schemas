- **Small enough** to move fast (population ~290,000)

- **Innovation-forward** with digital transformation leadership

## IX. Technical Architecture (Barbados Deployment)

### Infrastructure Stack

**Cloud Platform**: Azure (Government Cloud preferred) or GCP  
**Database**: PostgreSQL (hosted)  
**API**: NestJS backend at `/api/v1/*`  
**Frontend**: Next.js dashboards (youth, captain, ambassador, director, government)  
**Authentication**: JWT with role-based access control

- Barbados Data Protection Act compliance
